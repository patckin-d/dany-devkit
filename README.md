# Dany DevKit

Базовые конфигурации для разработки от Dany Pa.

## Особенности

- Конфигурация TypeScript
- Конфигурация ESLint с поддержкой Vue
- Конфигурация Prettier
- Правила Git ignore
- NPM скрипты для сборки

## Установка

1. Клонируйте репозиторий
2. Установите зависимости:
   ```bash
   npm install
   ```

## Использование

### Сборка

Для сборки проекта:
```bash
npm run build
```
